TUGAS 3

  1. Apa kepanjangan dari XAMPP?
  2. Bagaimana cara untuk mengakses XAMPP Panel?
  3. Module apa yang harus diaktifkan untuk menjalankan MYSQL Server?
  4. Bagaimana cara menjalankan PHPMyAdmin di browser?
  5. Bagaimana menjalankan MYSQL Client?

Jawaban

  1. XAMPP kepanjangan dari Cross Platform Apache MySQL/MariaDB PHP Pearl.
  2. Dengan membuka aplikasinya di START->All Programs->XAMPP->XAMPP Control Panel
  3. Module MySQL harus dijalankan pada XAMPP control panel.
  4. Dengan mengetikkan localhost:phpmyadmin pada alamat web browser.
  5. Buka command prompt lalu ketikkan "c:\xampp\mysql\bin\mysql -u root"
